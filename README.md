# P300-Transformer

ЭЭГ-Трансформер, основанный на Spatial-Temporal Tiny Transformer (S3T,  arxiv.org/abs/2106.11170), для многоклассовой классификации 16 символов на матрице P300.

explore_data - функции для создания синтетических данных из оригинальных
models - логистическая регрессия (baseline), minimal_S3T
preprocessing - функции подготовки данных (нормализация, склеивание)
